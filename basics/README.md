# Shell Basics

A collection of shell scripts covering fundamental Bash/shell commands.


## Scripts

- **0-current_working_directory**: Prints current absolute path
- **1-listit**: List items in current directory folder
- **2-bring_me_home**: Changes the active directory folder to the home 
- **3-listfiles**: List current directory folder in long format   
- **4-listmorefiles**: List current directory folder in long format with hidden items
- **5-listfilesdigitonly**: List current directory folder in long format with hidden items
- **6-firstdirectory**: Creates a directory inside `/tmp/my_first_directory/`
- **7-movethatfile**: Moves the file `/tmp/betty` to `/tmp/my_first_directory/`
- **8-firstdelete**: Deletes `/tmp/my_first_directory/betty`
- **9-firstdirdeletion**: Deletes `/tmp/my_first_directory/`
