# Shell Basics

A collection of shell scripts covering fundamental Bash/shell commands.

### Quick Use
```git update-index --chmod=+x [file]```
``` git add . && git commit -m 'work in basic t2' && git push ```

## Scripts

- **0-current_working_directory**: Prints current absolute path
- **1-listit**: List items in current directory
- **2-bring_me_home**: Changes the active directory to the home
